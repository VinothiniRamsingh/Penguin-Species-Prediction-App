3. This app demonstrates how machine learning models can be trained, deployed, and used interactively for predictions in a simple and intuitive way.

   # Key features:
   1. Island (categorical)
   2. Bill length (mm)
   3. Bill depth (mm)
   4. Flipper length (mm)
   5. Body mass (g)
   6. Sex (Male/Female)
   7. Species (target variable)
